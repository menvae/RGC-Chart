# RGC Chart
A Rust library for converting maps and charts between various rhythm games, with support for JavaScript and TypeScript via WebAssembly (WASM) for the Web and Node.js.

## Installation (Rust)
add this is your ``Cargo.toml``
```toml
[dependencies]
rgc-chart = "0.0.1"
```
or run
```
cargo add rgc-chart
```
